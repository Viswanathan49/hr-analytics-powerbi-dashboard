## 📌 Project Overview

This Power BI report was developed as part of the **Microsoft Elevate Power BI Internship 2024**, a 4-week virtual program. The dashboard provides comprehensive HR analytics covering employee salary, attendance, team performance, and workforce distribution.

## 📊 Visuals Included

### Page 1 – Salary Analytics
- **KPI Cards** — Total Salary, Max/Min/Avg Salary, Employee Count
- **Top 5 Employees by Salary** — Bar chart
- **Avg Salary by Team** — Comparative bar chart across 5 teams
- **Avg Salary by Job Level and Team** — Matrix table with conditional formatting
- **Bonus vs Salary Correlation** — Scatter plot with team-wise color coding

### Page 2 – Employee Data Report
- **KPI Cards** — Total Enrolled, Present, Absent, Avg Attendance, Avg Salary
- **Team Wise Attendance** — Stacked bar (Present vs Absent per team)
- **Manager Wise Attendance** — Stacked bar (Present vs Absent per manager)
- **Salary Distribution by Gender** — Donut chart (Male 62.66% / Female 37.34%)
- **Employees by Job Level** — Treemap (Analyst, Senior Analyst, Executive, Senior Executive, Team Lead)

---

## 🔍 Filters & Slicers

All visuals are interactive and respond to the following slicers:
- **Select Team** — Teams 1–5
- **Gender** — Male / Female
- **Senior Manager**
- **Job Level**
- **Reset Slicers** button for quick filter clearing
